What's an implicit surface

An implicit surface is a 3d shape defined by an equation F(x, y, z) = 0. Implicit surfaces offer easy CSG operations and can be far more compact than traditional 3D shape representations, which model shapes using individual triangle and point data.

What's Mermaid

Mermaid is an implicit surface renderer built with Zig and WebGPU. The core implementation is based off Matthew J. Keeter's paper Massively Parallel Rendering of Complex Closed-Form Implicit Surfaces. This project wouldn't be possible without his work, thank you Matthew!
